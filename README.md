# Registro de Estudiante en C#

Este proyecto es un programa en **C#** que permite registrar y mostrar información básica de un estudiante, incluyendo su ID, nombres, apellidos, dirección y números de teléfono.

---

## **Características**

- Implementación en **C#** usando programación orientada a objetos (POO).
- Clase `Estudiante` que encapsula atributos y métodos para manejar la información del estudiante.
- Uso de **arreglos** para almacenar múltiples números de teléfono.
- Validación de entradas numéricas para el ID del estudiante.
- Manejo de entradas vacías para evitar errores por valores nulos.
- Método `MostrarDatos()` para presentar toda la información ingresada de manera ordenada en la consola.
- Interacción con el usuario mediante la consola para ingresar los datos.
